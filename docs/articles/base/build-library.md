## 流程

在packages中开发，在example中查看，在docs中展示

## 基础

模块包管理：lerna
```
npm i -g lerna
```

组件设计：按需加载、扩展性延展性

单元测试：mocha

持续集成：shell、nodejs、git

代码规范、提交规范：vscode工具集成

## 组件库

区分：Vue2  Vue3  基于UI框架

打包：rollup

组件库文档工具：VitePress、自己实现

本地开发调试：
    框架库等依赖推荐使用CDN引入，不用额外配置
