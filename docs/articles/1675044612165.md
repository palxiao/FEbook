> 输出什么

```js
var a = { n: 1 }
var b = a
a.x = a = { n: 2 }

console.log(a.x)
console.log(b.x)
```

<details>
<summary> 查看答案 </summary>
    
    undefined 、{n: 2}

这道题主要是运算符优先级的问题，赋值结合性是从右向左的，但是这里的 a.x 优先级更高，所以是从左向右：
    
    a.x = a // 此时 a 为 { n: 1, x: { n: 1, x: .... } } 无限套娃
    a = { n: 2 } // 此时 a 为 { n: 2} 所以 a.x 就不存在了
    
</details>
