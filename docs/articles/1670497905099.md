### 实现 compose 函数

假设现有三个函数分别为 `a()` `b()` `c()`，请用一个函数实现 `c(b(a()))` 这种调用效果：

1. 数组出栈循环执行

```js
function compose(...funcs) {
  return function(result) {
    while(funcs.length > 0) {
      result = funcs.pop()( result )
    }
    return result
  }
}
```

结果：

```js
console.log( c(b(a(1))) === compose(a, b, c)(1) ) // true
```

2. 利用 reduce 实现

```js
function compose(...funcs) {
    return funcs.reduce((a, b) => (...args) => b(a(...args)))
}
```

3. 变种：实现 compose([a,b,c], init)，通过传入数组，init为初始调用值

可以在上面封装的 `compose` 函数下再封装一个函数即可：

```js
function compose2(arr, init) {
    return compose(...arr)(init)
}
```
